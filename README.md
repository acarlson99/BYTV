# BetterYTV

Better YouTube, inspired by [BetterTTV](https://betterttv.com/)

## Dependencies

- npm (version 7.8.0)
- node (12.20.1)
- http://browserify.org/

See `package.json` for version info

## Build

```
npm install
make
```

## Install

`chrome://extensions` -> enable dev mode (if disabled) -> load unpacked
